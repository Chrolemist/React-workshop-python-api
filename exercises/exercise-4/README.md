# Exercise 4: Fetch Users from API

**Time: 20 minutes**

## Requirements
1. Use useEffect to fetch on mount
2. Fetch from: https://jsonplaceholder.typicode.com/users
3. Show loading spinner
4. Handle errors
5. Type API response

## Tips
- Use async/await
- Three states: loading, error, success
- Check response.ok
- Empty dependency array []
