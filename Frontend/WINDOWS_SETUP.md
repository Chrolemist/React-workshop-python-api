# Windows Development Environment Setup

Complete guide for Windows users. See the full version in the workshop materials.

## Quick Steps

1. Install Node.js from https://nodejs.org/
2. Install VS Code from https://code.visualstudio.com/
3. Install Git from https://git-scm.com/
4. Clone this repository
5. Run `npm install`
6. Run `npm run dev`

For detailed instructions with screenshots, see the full WINDOWS_SETUP.md guide.
