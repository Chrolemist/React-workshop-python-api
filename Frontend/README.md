# React + TypeScript Workshop

Welcome to the React + TypeScript Workshop! This repository contains the starter code and exercises for building API-connected React applications with TypeScript.

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Code editor (VS Code recommended)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`
