# Exercise 1: Build a UserCard Component

**Time: 10 minutes**

## Requirements
1. Create a UserCard component in `src/components/UserCard.tsx`
2. Accept props: name, email, avatar (optional)
3. Define TypeScript interface
4. Display user information
5. Style as a card

## Test Data
```tsx
const testUser = {
  name: "Jane Smith",
  email: "jane@example.com",
  avatar: "https://i.pravatar.cc/150?img=1"
};
```

## Tips
- Use TypeScript interfaces
- Optional props with `?`
- Provide default avatar
